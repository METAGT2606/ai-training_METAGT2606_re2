# docs

このディレクトリには、最終課題の設計メモや評価結果を格納します。

## 格納するドキュメント（受講者が作成）

### 1. 状態遷移図

LangGraphの状態遷移を図示したファイル（例: `state-diagram.md` or `state-diagram.png`）

記載内容：
- stateの定義（保持する変数）
- ノードの一覧と役割
- エッジの条件

### 2. 主要プロンプト

エージェントで使用するプロンプトの設計メモ（例: `prompts.md`）

記載内容：
- システムプロンプト
- 分類用プロンプト
- 回答生成用プロンプト

### 3. 失敗パターンと対策

想定する失敗パターンと対策のメモ（例: `failure-patterns.md`）

記載内容：
- 失敗パターン（例: JSONパース失敗、検索ヒットなし）
- 対策（リトライ、フォールバック、エラーメッセージ）

### 4. 評価結果と改善ログ

テスト実行の結果と改善履歴（例: `evaluation.md`）

記載内容：
- テストケース一覧
- 成功/失敗の集計
- 改善前後の比較
